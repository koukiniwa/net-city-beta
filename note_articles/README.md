## 投稿前にやること

### 1. 記事を読み直して、修正する
- 「ここ、違う」と思ったら変える
- 「この表現の方が自分らしい」と思ったら変える
- **あなたの言葉に、する**

### 2. 追加したいエピソードがあれば、入れる
- 開発中の具体的な失敗談
- 完成した時の感情
- 友達や家族の反応

### 3. 画像を用意する
- アプリのスクリーンショット（2-3枚）
- サムネイル画像

### 4. Twitterアカウントのリンクを入れる
- 記事の最後に追加
